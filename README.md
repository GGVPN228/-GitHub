## Описание
**GitHub User Finder** - это графическое приложение на Python для поиска пользователей GitHub, просмотра их профилей и сохранения избранных пользователей в локальном JSON-файле.

### Возможности:
- 🔍 Поиск пользователей по имени
- 📋 Отображение результатов поиска с количеством репозиториев
- ⭐ Добавление/удаление пользователей в избранное
- 💾 Автоматическое сохранение избранных в JSON
- 👁️ Детальный просмотр профиля пользователя
- 📊 Статистика (количество подписчиков, репозиториев и т.д.)

## Как использовать API

### GitHub REST API
Приложение использует следующие эндпоинты GitHub API:

#### 1. Поиск пользователей
- `q` - поисковый запрос (имя пользователя)
- `per_page` - количество результатов на страницу (макс. 100)

#### 2. Получение информации о пользователе
Возвращает детальную информацию:
- `login` - имя пользователя
- `name` - полное имя
- `public_repos` - количество публичных репозиториев
- `followers` - количество подписчиков
- `following` - количество подписок
- `bio` - биография
- `html_url` - URL профиля

### Ограничения API:
- **Без аутентификации:** 60 запросов в час
- **С аутентификацией (токен):** 5000 запросов в час
